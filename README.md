# 📈 DemandForge AI - Time Series Forecasting Platform

**Advanced Time Series Forecasting with Uncertainty Quantification**

## Features
- Prophet Model with Prediction Intervals (Uncertainty)
- Interactive Forecasting Dashboard
- Anomaly Detection
- Model Comparison Ready
- Downloadable Forecast Reports

## 🎯 Objective
To predict future demand/sales with **confidence intervals** and detect anomalies in time series data.

## 🛠 Tech Stack
- **Model**: Facebook Prophet
- **Frontend**: Streamlit
- **Visualization**: Plotly
- **Uncertainty**: Built-in Prediction Intervals

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
